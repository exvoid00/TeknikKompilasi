############################################################
#  [15:30]  BAGIAN 7 - TAHAP 6: CODE GENERATION            #
############################################################

   >>> BUKA FILE: code_generator.py

BACA:
"Tahap terakhir adalah Code Generation. Compiler mengubah AST
menjadi kode antara berbentuk THREE-ADDRESS CODE, atau kode
tiga alamat."

   TUNJUK baris 16 (new_temp), baris 20 (new_label), baris 89 (visit_If), baris 105 (visit_While)
BACA:
"Di kode, saya membuat fungsi untuk menghasilkan variabel
sementara seperti t1 dan t2, serta label seperti L1 dan L2.
Untuk percabangan dan perulangan, diterjemahkan menjadi
perintah lompat IF_FALSE dan GOTO, mirip cara kerja bahasa
mesin."

   (kembali ke terminal, lihat output bagian [6])

   CONTOH OUTPUT (Three-Address Code):
      pesan = "Apo kaba"
      PRINT pesan
      x = 14
      READ x
      t1 = x > 10
      IF_FALSE t1 GOTO L1
      PRINT "gadang"
      GOTO L2
      L1:
      PRINT "ketek"
      L2:
      i = 0
      L3:
      t2 = i < 3
      IF_FALSE t2 GOTO L4
      PRINT i
      t3 = i + 1
      i = t3
      GOTO L3
      L4:

BACA:
"Inilah hasilnya. Pemberian nilai menjadi 'x sama dengan 14',
input menjadi READ, cetak menjadi PRINT. Untuk percabangan:
dihitung t1 sama dengan x lebih besar 10, lalu IF_FALSE t1 GOTO
L1, artinya kalau salah lompat ke blok else. Inilah hasil akhir
compiler."
